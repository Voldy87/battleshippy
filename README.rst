Introduction
==============
This a battleship game: it is available in different shapes:
* a local Python executable installable that allows play against an AI or another player (sharing the same computer); graphical or terminal Interface
* a distributed application game (p2p or client-server), playing against AI or remote players
* a Django-based website with the same characteristics of the upper one
